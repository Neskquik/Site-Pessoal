# Nesk Developer

Página pessoal desenvolvida para apresentar o perfil **Nesk Developer**, reunindo links importantes, identidade visual personalizada e um player de música integrado em uma interface simples e moderna.

## 🔗 Acesse o projeto

[Ver site online](https://neskqzn.netlify.app/)

## ✨ Funcionalidades

* Página pessoal com visual personalizado
* Links para perfis e plataformas externas
* Player de música integrado
* Layout simples, direto e responsivo
* Hospedagem online via Netlify

## 🛠️ Tecnologias utilizadas

* HTML
* CSS
* JavaScript
* Netlify

## 📌 Objetivo

O objetivo deste projeto é criar uma página pessoal estilizada para centralizar links, apresentar a identidade do criador e oferecer uma experiência visual agradável para quem acessa o perfil.
